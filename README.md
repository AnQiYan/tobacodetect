<h5>目标：</h5>
输入一张包烟橱窗的照片，识别照片中的包烟并且统计数量

<h5>数据集：</h5>
在这个项目中，我们使用了商店售烟橱窗的照片作为输入数据集。橱窗照片包含了多种不同品牌的包烟。

<h5>数据预处理：</h5>
由于不同拍摄者和不同设备拍摄出的包烟橱窗照片尺寸大小不一。因此我们首先将数据做了预处理。将其大小调整为固定尺寸：宽为600px, 高为450px。将图片调整为固定尺寸之后，将使用标注工具labelimg标注图像。
